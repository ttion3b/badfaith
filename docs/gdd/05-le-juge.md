# Le Juge — l'unique arme

## Concept
Un seul revolver sur la map, une seule balle. Son but premier n'est pas de tuer : c'est un instrument de braquage au micro ("pose ton loot ou je tire") et de bluff permanent.

## Règles
- Spawn : emplacement semi-aléatoire parmi 5 spots connus, jamais annoncé. Le trouver est un avantage informationnel.
- Une balle. Tir = mort instantanée de la cible (sa Poche Perso va au Pot Commun, règle standard).
- **Vérifier le barillet est une action PRIVÉE** (animation discrète, seul le porteur voit le résultat). Personne d'autre ne sait jamais si l'arme trouvée/échangée/volée est encore chargée → le bluff survit au tir.
- Le tir est ENTENDU par toute la map (bang + annonce sonore de La Direction : "Un coup de feu a été tiré."). Mais pas le lieu exact ni le tireur.
- L'arme se pose, se vole (sur un joueur endormi/enfermé), s'échange, se donne en garantie d'une alliance.
- Braquage : aucun système mécanique — c'est du pur roleplay au micro. Le jeu fournit l'arme, les joueurs fournissent le théâtre.
- L'arme est visible dans la main (pas d'arme cachée au MVP) : la menace est un choix public.

## Interactions avec les autres systèmes
- Prime sur tête (Pacte 4) + Le Juge = le seul moyen "propre" d'encaisser une prime.
- Au Tribunal, le tir n'est PAS anonyme : la GoPro de la victime a tout filmé. Tirer est l'acte le moins déniable du jeu — c'est voulu : le coût du meurtre direct est social, pas mécanique.
- Post-MVP : une deuxième balle cachée quelque part dans la map (très rare), pour casser la certitude "l'arme est vide, il a déjà tiré".
