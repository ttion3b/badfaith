# MAUVAISE FOI (BAD FAITH)

Jeu multijoueur 4-8 joueurs de trahison sociale à déni plausible, conçu pour la viralité Twitch/TikTok.
Coopérez pour survivre. Acceptez des Pactes secrets pour trahir. Mentez au Tribunal. Tout est filmé.

- **Game Design Document** : [docs/gdd/](docs/gdd/) — commencer par [00-vision.md](docs/gdd/00-vision.md)
- **Roadmap** : [docs/gdd/10-roadmap.md](docs/gdd/10-roadmap.md)
- **Stack** : Unity 6 (URP, rendu PSX), FishNet + Steam Datagram Relay, Dissonance (voice de proximité)
